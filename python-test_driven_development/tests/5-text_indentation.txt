The ``text_indentation`` function prints a text with 2 new lines after
each of these characters: ``.``, ``?`` and ``:``

>>> text_indentation = __import__('5-text_indentation').text_indentation

Basic test:
>>> text_indentation("Hello. How are you? I'm fine:")
Hello.
<BLANKLINE>
How are you?
<BLANKLINE>
I'm fine:
<BLANKLINE>

With extra spaces:
>>> text_indentation(" Hello. World? Yes: ")
Hello.
<BLANKLINE>
World?
<BLANKLINE>
Yes:
<BLANKLINE>

Empty string:
>>> text_indentation("")

Type error:
>>> text_indentation(12)
Traceback (most recent call last):
...
TypeError: text must be a string
